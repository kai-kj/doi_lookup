DOI Lookup

A simple DOI lookup tool. The main focus of the tool is to shown relationships (citations and references) between articles.

Article metadata and search functionality is provided by the Crossref REST API , relationships are provided by the OpenCitations Indexes unifying REST API , and citation text is taken from doi.org (see here for more info).

This website is written in vanilla JavaScript, and the source code is available on GitHub .
